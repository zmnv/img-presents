## How to use

1. Download repository directly from github or:
```
git clone https://github.com/zmnv/zmnv-gg.git
```

2. Install packages:
```
npm install
```

3. Prepare image data. Place your images in `images/` folder. You can group they by subfolders.

4. Run app menu to start:

```
npm start
```
